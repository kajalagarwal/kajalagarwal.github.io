What is JavaScript Used For?

Properly constructed web pages are built using up to three different languages.

  • HTML
  • CSS
  • JavaScript

HTML

The HTML defines what the content is. When written properly no attempt is made to define how that content is supposed to look. How it will look when used on different screens.

CSS

The appearance of a web page is defined using CSS which has the ability to specify which media the specific commands are to apply to so as to be able to have the content formatted appropriately for whatever device the page is being accessed with.

Javascript

The purpose of JavaScript is to solve this problem

It convert your static page into dynamic so that now user can interact with the page without waiting for the new page to load everytime he made changes.

JavaScript adds behaviour to the web page where the web page is capable of responding to actions by your visitors without needing to load a new web page in order to process their request
