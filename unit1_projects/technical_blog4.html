Array Vs Hashes


Array

Array is basically a block where you store the value.

For an example if you have put same color balls in one box then to differentiate it what you will do?
You will try to give them individuality by giving them a name and put them inside a box which has its own name which is what exactly happening in Array

Array name is supposed to be a name of a big box and element are the same color balls with their own names

Array elements can be anything string, integer, float etc

There can be different types of arrays

  1. Multidimensional
  2. Vertical
  3. Horizontal

Example Array

Array=[1,2,3,"new"]
Array[0]=1; Array[1]=2; Array[2]=3; Array[3]="new"

Hashes

Hashes are basically an updation of array.

It consist key value pair where key store the name of the envelope you wanna post and value is the value inside the envelope. Hash name is the post box where are the envelopes are present

Hashes are dependent on only keys not the position like array.

Example Hash

Hash={key1=>"value1",key2=>"value2"}
Hash[key1]="value1"; Hash[key2]="value2" ;

Hash rocket (=>) are used to show equals inside Hash function

Hash keys can be

  1. Symbol
  2. String
  3. Integer

Symbols are non changeable name used in hash as a key which means user can iterate on them but can't change their name.

Hash values can be

  1. String
  2. Integer

It can't take symbol as a value
